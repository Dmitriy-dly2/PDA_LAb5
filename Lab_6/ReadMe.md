# Python Labs Repository

В этом репозитории хранятся лабораторные работы по Python.

## Правила

* Каждая лабораторная работа должна находиться в отдельной директории (`lab_1`, `lab_2`, и т.д.)
* Внутри каждой лабораторной должны быть:

  * исходный код
  * краткое описание задания (README.md)
